ADB的使用

R.java文件（“Android app 资源字典”）

Android项目下，由AAPT工具根据应用中的资源文件自动生成的Java类
生成规则：

  • 每类资源对应R类的一个内部类。例如界面布局资源对应的layout内部类。
  • 每个具体资源项都对应内部类的一个public static final int 类型字段。

Android清单文件AndroidManifest.xml

Android应用全局描述文件，说明了该app的名字、使用的图标、包含的组件等
主要包含如下信息：

  • 当前应用程序的包名，该包名为该应用的唯一标识
  • 当前应用程序包含的组件，如Activity、Service、BroadcastReceiver和ContentProvider等
  • 当前应用所兼容的最低版本号
  • 当前应用所使用系统所需的权限声明
  • 其他程序访问该应用所需的权限声明

部署到Github

步骤：

  1. 安装Git
  2. 注册Github账号
  3. 在Android studio中设置Git的安装路径：
    点击 Android studio -》preferences-》Git 选择git 的安装目录，并点击右侧的test，test成功就行。
    (一般在使用macOS的情况下，Android Studio会自动找到路径)
  4. 配置Github账号:
    点击add acoount，会弹出你需要输入GitHub的账户和密码，输入完账户密码后，Android Studio 会自动连接GitHub。
  5. 上传项目
    进入到Android Studio中后，在菜单栏中选择VCS -》import into version control -》share project on GitHub
